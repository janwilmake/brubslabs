My Story

Hey there! I'm Jan, and for the last 3 years, I've been all in on building an army of AI agents to scale productivity. Back in April 2022, I took the leap and quit my job to dive into full-time independent research on applied AI.

My main focus? Creating an AGI-like system by building software that can reliably build software. I've been working on open-source, high-scale AI code analysis and transformation tools that make development faster and more efficient.

When I'm not coding, I'm exploring ways to integrate AI into everyday workflows and building tools that help developers be more productive. My goal is to create a future where AI and humans work seamlessly together, multiplying what we can accomplish.
